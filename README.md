# Employee Payroll System

## 📌 Overview

The Employee Payroll System is a Java desktop application developed using **Java Swing**, **JDBC**, and **MySQL**. It helps manage employee records and automates payroll calculations through a user-friendly graphical interface.

## ✨ Features

* Add new employee details
* Search employee by Employee ID
* Update employee information
* Delete employee records
* Display all employee details
* Automatic salary calculation
* Simple and easy-to-use GUI
* MySQL database integration

## 🛠️ Technologies Used

* Java
* Java Swing
* JDBC
* MySQL
* SQL

## 💰 Salary Calculation

**Gross Salary = Basic Salary + HRA + DA**

Where:

* **Basic Salary** – Fixed salary of the employee
* **HRA (House Rent Allowance)** – Housing allowance
* **DA (Dearness Allowance)** – Cost-of-living allowance

## 🗄️ Database

The project uses MySQL to store employee information securely.

Example employee details:

* Employee ID
* Employee Name
* Department
* Designation
* Basic Salary

## 🚀 How to Run

1. Install Java (JDK).
2. Install MySQL.
3. Create the required database and employee table.
4. Update the database connection details in the project.
5. Compile and run the Java application.
